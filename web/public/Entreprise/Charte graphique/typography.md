Polices recommandées

Usage web et documents :
- Police UI / corps de texte : `Inter` ou `Roboto` — lisible à l'écran, bonne neutralité.
- Police titres : `Merriweather` ou `Montserrat` — pour donner du caractère aux titres.

Hiérarchie typographique conseillée :
- Titre principal (H1) : 36–48 px, poids semi‑bold/700
- Titre secondaire (H2) : 24–32 px, poids 600
- Corps de texte : 16 px, poids 400
- Légendes / notes : 12–14 px

Règles :
- Ne pas utiliser plus de deux familles de polices simultanément.
- Préférer les variantes variable/woff2 pour le web pour des temps de chargement réduits.
