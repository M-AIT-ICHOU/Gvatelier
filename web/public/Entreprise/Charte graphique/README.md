Charte graphique — Atelier

But : centraliser les règles visuelles et les assets officiels de l'entreprise.

Contenu du dossier :
- `assets.md` : inventaire des fichiers de logo et organigramme existants.
- `colors.md` : palette de couleurs recommandée (hex) et usages.
- `typography.md` : polices recommandées et hiérarchie typographique.
- `usages.md` : règles d'utilisation du logo (espacement, tailles minimales, fonds, interdits).

Procédure :
- Pour ajouter un nouveau logo ou variante, déposer le fichier dans `Entreprise/Logo` puis mettre à jour `assets.md`.
- Les fichiers images (JPEG, SVG) sont suivis dans le dépôt ; si certains dépassent la limite, utiliser Git LFS.
