Palette recommandée

Couleurs principales :
- Vert forêt (primaire) : #145214 — usage : logo principal, titres, éléments identitaires.
- Vert olive (secondaire) : #6B8E23 — usage : fonds secondaires, accents.
- Orange accent (signalement) : #D9534F — usage : boutons d'appel à l'action, avertissements.

Neutres :
- Gris foncé (texte) : #333333
- Gris clair (fond) : #F4F4F4
- Blanc : #FFFFFF

Règles :
- Respecter un contraste suffisant pour la lisibilité (WCAG AA) : texte sur fond doit respecter un ratio adéquat.
- Utiliser la couleur primaire pour l'identité visuelle ; la couleur accent est réservée aux éléments de mise en valeur.
- Pour l'impression, préférer les équivalents CMJN : Vert forêt ≈ C:100 M:5 Y:100 K:60 (vérifier en print proof).
